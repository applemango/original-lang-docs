IDE

Ctrl + Shiftkeyでconsoleとeditorを行き来できる

consoleで一行だけコードを実行できる

基本

"#"はコメントです必ず最初につけてください

改行やインデントは一部変更することが可能です


#test
var x = 10
var y = 50
if(x==10)
{
if(y==50)
{
console.log(xは10yは50)
}
}

上記のコードは以下のように変更できます


var x = 10
var y = 50
if((x==10)&&(y==50))){
    console.log(xは10yは50)
}

更にこのように変更できます


var x = 10;var y = 50;if((x==10)&&(y==50))){console.log(xは10yは50)}
